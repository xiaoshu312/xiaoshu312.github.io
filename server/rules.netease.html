飘零889的网易租赁服规则

社交 · 建筑 · 操作员守则 · 处理方式

1. 社交

  1. 不能乱打人

  2. 不能在聊天框中随意辱骂他人

  3. 严禁破坏建筑

  4. 严禁杀村民和别人的宠物（如别人的马，狗，猫等等，尤其是马）

  5. 严禁偷东西

  6. 如果未经过主人同意不能骑别人马（除非是帮别人找到了丢失的马）

2. 建筑

  1. 不能破坏地形（如乱在地上放东西、挖坑）

  2. 建房子尽量不用占用公共空间

3. 操作员守则

  1. 不开创造或使用指令作弊拿取物资（除修复服务器或理赔或建筑）

  2. 给予他人操作员需服主同意

  3. 不乱使用指令

  4. 降低他人权限等级需经过服主同意

4. 处理方式

  1. 违反规则轻则提醒、修复、赔偿，重则关闭种种权限或直接拉黑

  2. 违反规则后死不承认后被发现必然拉黑

操作员守则：违反守则按情节判断撤销权限或直接封禁
